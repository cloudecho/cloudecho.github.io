I created /Library/LaunchDaemons/local.localhost.startup.plist containing the code below. It runs the rc.local script once at start up.





    Label             local.localhost.startup
    Disabled          
    RunAtLoad         
    KeepAlive         
    LaunchOnlyOnce    
    ProgramArguments
        
            /etc/rc.local
